# ReUnited-Frontend

Blazor frontend with Razor components for the ReUnited application.

ReUnited is a lost-and-found application designed to help match people with their lost belongings.

## Tech Stack

- C# / .NET 8
- Blazor
- Razor components
- Bulma

## Prerequisites
- Visual Studio 2022
- .NET 8 SDK

## Instructions

Clone this repository, then open the `root/ReUnited-Frontend` directory. Ensure the backend is also running, then run:

```
dotnet run --project ReUnited-frontend
```
